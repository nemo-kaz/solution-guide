      InternetGateway

      パターン

      説明

      OCIでは、不要なインスタンスのweb公開を避けるため、標準状態ではインスタンスが外からのアクセスを受け付けない状態となっている。インターネットからの接続を受け付けるためには、インスタンスに紐付いたVirtualCloudNetwork(VCN)にInternetGatewayを作成する。

      用途

      (LoadBalancerを用いない)Webサーバーなど、外部から直接アクセスを受け取ることができる。
